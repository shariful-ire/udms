AI-Assisted Surgical Navigation Robot for Low-Resource Hospitals

Action: Developed a 4-DOF robotic arm with ROS integration, a computer vision module using OpenCV and TensorFlow for real-time instrument tracking, and a Python-based control interface. Conducted 50+ simulated surgical scenarios in Gazebo. Result: Achieved 94% accuracy in simulated tissue targeting; prototype selected for university innovation showcase.
